MDI
PROG.
M3 (sentido de rotação) S1000 (rotação) M7 (liga lubrificação) EOB INPUT.
LIGA MAQUINA (botão verde)
APERTA 2X HND, SELECIONA O SENTIDO PARA ONDE VAI A FERRAMENTA E EM SEGUIDA DESCE ATÉ CORTA A PEÇA.
SELECIONAR PROGRAMA
PROG. → PASTA → SELECIONAR O PROGRAMA DESEJADO → MEN → OPRT →PRINCIPAL.
2X OFFSET → TRABALHO.
G54 =98 → INPUT → 118
MAGAZINE
F1 → F1 → F2 → COLOCAR O COMPRIMENTO DESEJADO DA PEÇA TOTAL → ENTER.
ZERAR FERRAMENTA
ENCOSTAR A PEÇA NA CASTANHA
FECHAR A PORTA → LIGA AUTOMÁTICO DO MAGAZINE → LIGA A MAQUINA NO AUTOMÁTICO (como se fosse fazer a 1° peça).
QUANDO TERMINAR DE PUXAR A PEÇA, PARE A MAQUINA NO BOTÃO RESET.
ZERAR A FERRAMENTA NO EIXO Z.
2X OFFSET → GEOMETRIA →SELECIONA G007 → E COLOQUE NO EIXO Z.
VÁ COM A FERRAMENTA ATÉ PASSAR DO FACEAMENTO DA PEÇA E COLOQUE O CALÇO DE PRECISÃO ATÉ A FERRAMENTA FACEAR COM O PEÇA.
Z0 → MEDIR.
CASO O EIXO X NÃO ESTEJA ZERADO FAÇA O MESMO PROCEDIMENTO
X(mais o diâmetro externo da peça) → MEDIR.
PUXAR A PEÇA PARA ZERAR A FERRAMENTA DO INTERNO N7 → MEN (liga a máquina no automático até fazer o rebaixo).
ENCOSTAR A PONTA DA PASTILHA NA FACE DA PEÇA.
Z0 → MEDIR.

X(mais o diâmetro externo da peça) → MEDIR.

MEN → N5 → 2X LIGAR → DEIXAR A LUBRIFICAÇÃO DESLIGADA, OLHAR PARA SABER SE VAI FAZER O INTERNO NORMALMENTE.
LIGAR MAQUINA NO AUTOMATICO.
AJUSTE FERRAMENTA DIMENSIONAL
