# Mapa de Processo — Produção 2026

**Rev. 15 · aprovado 2026-02-10 por Fernando Macedo**

## Resumo executivo

Mapa do processo central da KAB — produção de tubos/perfis de aço e alumínio (corte, chanfro, prensas, ergomat). Gestor **Fernando Macedo** com 34+ atividades SIPOC cobrindo desenvolvimento de novos produtos, kick-off com QA, FMEA, compra de MP, treinamentos, programação semanal, emissão de OPs, setup, produção, apontamentos no DBCorp, manutenção corretiva, retrabalho/seleção e pedidos de compra. **2 indicadores chave**: Produção ≥100% do planejado E Refugo ≤0,50%. Riscos formais: peças NC recorrentes sem ação imediata, perda de ritmo. Oportunidade: controle de insumos (pastilhas, parafusos) pela própria produção.

## SIPOC — fluxo macro

1. **Desenvolvimento**: análise viabilidade (FOR-018) → kick-off → FMEA → compra amostra → levantamento treinamentos
2. **Compra de MP**: planilha programação → pedidos DBCorp → aprovação alta direção → envio fornecedor → recebimento
3. **Programação**: pedido cliente lançado (Suellen) → plano de entrega → programação máquinas → pré-setup → emissão OP
4. **Execução**: separação documentação (facilite/desenho/registro corte) → setup operador → produção → apontamento DBCorp
5. **Manutenção corretiva**: emissão OS → parada DBCorp → validação retorno
6. **Retrabalho/seleção**: análise produto NC → disposição → inspeção QA 100% → apontamento
7. **Pedido compra indireta**: orçamentos (Gabriel) → aprovação alta direção → DBCorp (IT-059)
