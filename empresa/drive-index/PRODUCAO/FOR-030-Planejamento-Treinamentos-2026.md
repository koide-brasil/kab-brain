# FOR-030 REV01 — Planejamento Anual de Treinamentos OPERADORES (2026)

**Arquivo MASTER de planejamento de treinamentos da Produção · modificado HOJE (20/05/2026)**

## Resumo executivo

Planilha-mestre do **FOR-030 versão 01** que planeja TODOS os treinamentos dos OPERADORES da KAB pra 2026, **mês a mês**. Estrutura: cada linha é um treinamento + responsável (RH, Qualidade, Produção, Manutenção, Ferramentaria), distribuídos em 12 colunas (jan-dez) com status **P** (Previsto) / **O** (Realizado) / **R** (Reprogramado) / **X** (Não Realizado). **Status atual**: FEV/2026 com forte volume de treinamentos realizados (maioria "O"). MAI/2026 com apenas 1 treinamento (IT-062). **Cruza com daily-meeting aba RH** que consome FOR-030 pra exibir adesão mensal.

## Estrutura

| Coluna | Conteúdo |
|---|---|
| Item | Tipo (PRODUÇÃO) |
| Nível | BÁSICO / INTERMEDIÁRIO / AVANÇADO |
| Descrição | Nome do treinamento (geralmente IT-XXX) |
| Responsável | RH / Qualidade / Produção / Manutenção / Ferramentaria |
| Realizados | Tipo (Interno / Externo) |
| JAN a DEZ | Status P/O/R/X por mês |

- **O** = Realizado ✅
- **R** = Reprogramado
