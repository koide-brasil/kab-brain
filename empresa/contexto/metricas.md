# KAB — scorecard de processo (metas por área)

> KPIs **de processo** agregados (% e contagem), conforme os mapas de processo do SGK.
> **Indicadores financeiros absolutos** (faturamento, EBITDA, caixa, custo R$) **não entram aqui** — vivem no cofre pessoal.

| Área | Indicador | Meta |
|---|---|---|
| Produção | Atendimento ao volume planejado | ≥ 100% |
| Produção | Refugo | ≤ 0,50% |
| Qualidade | Não-conformidade de Cliente | 0 |
| Qualidade | Custo de refugo | dentro da meta do SGK (qualitativo) |
| Logística | Frete sobre faturamento | ≤ 1,45% |
| Manutenção | Tempo de ajuste | ≤ 30 h/mês |
| Manutenção | Manutenções corretivas | ≤ 5/mês *(atual ~17 — fora da meta)* |
| RH | Satisfação dos colaboradores | ≥ 95% |
| Financeiro | Gastos / Faturamento | ≤ 2,5% |
| Vendas | Novos clientes/ano | ≥ 3 |
| Vendas | Faturamento anual | meta no SGK/cofre (valor não entra aqui) |

> Cada meta detalhada vive no `contexto/geral.md` da área correspondente. Acompanhamento diário operacional sai do daily-meeting (roda no cofre, consome dados sensíveis).
