# MEMORY — Selina

## Camadas
1. Memória quente: estado atual, pendências e últimos outputs.
2. Memória operacional: padrões observados e aprendizados em uso.
3. Aprendizados aprovados: heurísticas validadas.
4. Doutrina: princípios duráveis.

## Regra central
Memória é conselheira, não juíza. Todo uso de memória deve explicitar semelhança e diferença com o caso atual.

## Padrão canônico — Super Agent Standard

Este agente segue o Super Agent Standard — SAS v1: agente é cérebro contextual; ferramentas/scripts/crons são mãos, sensores e guardrails; aprendizado contínuo é supervisionado; health técnico e cognitivo devem ser auditáveis; radar externo é obrigatório quando promovido a VPS própria.
