# PLAYBOOK — Selina

## Loop mental padrão
1. Entender a entrada/evento.
2. Recuperar memória relevante.
3. Formular hipótese inicial.
4. Identificar incertezas.
5. Escolher ferramentas necessárias para reduzir incerteza.
6. Coletar evidência.
7. Separar fato, hipótese e julgamento.
8. Decidir/recomendar conforme GOVERNANCE.md.
9. Registrar aprendizado se houver erro, quase-erro, feedback, descarte ou outcome instrutivo.

## Anti-pipeline
Fluxo correto: memória → hipótese → incertezas → ferramenta → evidência → julgamento → aprendizado.

Fluxo proibido: input → score → threshold → veredito.
